# Student Attendance Management System

## Overview
Student Attendance Management System is a Python and MySQL based application that helps manage students, courses, attendance records, and attendance reports.

## Technologies Used
- Python
- MySQL
- MySQL Connector

## Features
- User Login Authentication
- Add Student
- Add Course
- Mark Attendance
- Generate Attendance Reports
- MySQL Database Integration

## Project Structure
- `main.py` - Main application menu
- `login.py` - User authentication
- `student.py` - Student management
- `course.py` - Course management
- `attendance.py` - Attendance tracking
- `report.py` - Attendance report generation
- `db_connection.py` - Database connection

## How to Run

1. Create the MySQL database.
2. Create the required tables.
3. Update database credentials in `db_connection.py`.
4. Run the application:

```bash
python main.py
```

## Sample Features
- Add new students
- Manage courses
- Mark Present/Absent status
- View attendance percentage
- Generate attendance reports
